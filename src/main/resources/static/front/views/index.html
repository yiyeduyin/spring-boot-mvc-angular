公司简介

信詠塑胶工业股份有限公司于西元1982年成立于台湾省台北县三重市,专业从事塑胶、电木模具设计、开发、制造及射出成型， 主要产品为BOBBIN、BASE、CASE、SMD等专业变压器骨架、电子塑胶配件。公司致力于塑胶注塑产业专业化的生产与经营，不断提高产品品质与产能， 全面推行ISO9001管理体系，严格执行ROSH等各项环保规定，获得UL认证及多项国际认证；因应市场发展及扩张需求，西元1996年, 于中国广东省东莞市成立东莞信永电子科技有限公司；西元1998年,于中国江苏省吴江市成立苏州正杰塑胶电子有限公司； 西元2005年,于中国广东省东莞市成立东莞市烨坤塑胶电子有限公司；逐步形成注塑行业集团式发展， 成为塑胶电子行业中带来高品质、高效率的楷模典范；经过数十年的积极开拓，公司取得塑胶电子行业中500家以上客户的信赖与支持， 行销客户涵盖美国、日本、中国等世界各地电子产品百大制造商，产品类别遍布车载电子、家电、网路、娱乐电子产品等上千种终端产品， 主要合作供应商有日本SUMITOMO(住友)... 更多
