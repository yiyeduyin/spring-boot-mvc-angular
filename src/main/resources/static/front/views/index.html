公司简介

信咏塑胶工业有限公司（信咏塑胶电子工业）是一家专业从事塑胶、酚醛模具和R/D制造 业的公司。我们致力于bobbin开发和生产已20多年,赢得了200多客户的评估和支持。我们通 过了ISO9001(NO.5773)和ISO14001(NO.ENV318)的认证,并真正将ISO14001标准落实 到强烈实践中。为了提供高质量的产品给我们的客户,我们细心处理好每一个订单、站在客户角度 上思考，有竞争力的价格和高质量的服务成为我们的客户再次将订单给我们的原因。信咏总是渴 望抓住一切机会用感激和尊重的心回报我们的客户。信咏塑料工业有限公司有限公司成立于1982 年6月,在台北城市圣钟。信咏塑料电子工厂建于丁伟镇,东莞市1996年10月。苏州正杰塑料电子 有限公司建于1998年。我们公司一开始是用原材料制造OEM电子元件但渐渐地我们投入产品开发 并转型成一个有创新设计能力的公司,生产订制产品仍然是我们的核心业务。信咏越来越高效做各 种不同的塑料制品,我们为来自各行各业的客户提供最好的产品。
